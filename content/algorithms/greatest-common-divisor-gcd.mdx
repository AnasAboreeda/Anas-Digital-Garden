# Greatest Common Divisor [GCD]

Let a′ be the remainder when a is divided by b, then
`gcd(a, b) = gcd(a′, b) = gcd(b, a′)`

```java
public class GCD {
  public static int gcd_naive(int a, int b) {
    int current_gcd = 1;
    for (int d = 2; d <= a && d <= b; ++d) {
      if (a % d == 0 && b % d == 0) {
        if (d > current_gcd) {
          current_gcd = d;
        }
      }
    }

    return current_gcd;
  }

	// Euclidean Algorithm --- O(log n)
  public static int gcdFast(int a, int b) {
    if (b == 0) {
      return a;
    }

    return gcdFast(b, a % b);
  }
}
```
