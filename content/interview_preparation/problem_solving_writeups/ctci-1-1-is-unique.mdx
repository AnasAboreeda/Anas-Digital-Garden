# CTCI 1.1. Is Unique


## Problem Statement

## Solution

```java
public class IsUnique {

  public static boolean checkIsUnique(String str) {
    Map<Character, Boolean> found = new HashMap<>();
    for (char c : str.toCharArray()) {
      if (found.containsKey(c)) {
        return false;
      }
      found.put(c, true);
    }

    return true;
  }

  public static void main(String[] args) {
    System.out.println("checkIsUnique(abc)  true == " + checkIsUnique("abc"));
    System.out.println("checkIsUnique()  true == " + checkIsUnique(""));
    System.out.println("checkIsUnique(aaa)  false == " + checkIsUnique("aaa"));
  }
}
```
