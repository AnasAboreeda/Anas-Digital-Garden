# Template Method

## Uml

![None](../umls/template-method.png)

## Implementation

1. Base class

```java
public abstract class ComputerManufacturer {

	public void buildComputer() {
		addHardDesk();
		addRam();
		addKeyboard();
	}

	public abstract void addKeyboard();

	public abstract void addRam();

	public abstract void addHardDesk();

}
```

1. implementations

```java
public class DesktopManufacturer extends ComputerManufacturer {

	@Override
	public void addKeyboard() {
		System.out.println("adding Desktop Keyboard");

	}

	@Override
	public void addRam() {
		System.out.println("adding Desktop Ram");

	}

	@Override
	public void addHardDesk() {
		System.out.println("adding Desktop Harddesk");

	}

}
```

&&

```java
public class LaptopManufacturer extends ComputerManufacturer {

	@Override
	public void addKeyboard() {
		System.out.println("adding laptop Keyboard");

	}

	@Override
	public void addRam() {
		System.out.println("adding laptop Ram");

	}

	@Override
	public void addHardDesk() {
		System.out.println("adding laptop Harddesk");

	}

}
```

1. Test

```java
public class Test {

	public static void main(String[] args) {
		ComputerManufacturer desktop = new DesktopManufacturer();
		desktop.buildComputer();

		ComputerManufacturer laptop = new LaptopManufacturer();
		laptop.buildComputer();
	}

}
```
