# 242. Valid Anagram

## Problem Statement

## Solution

```java
class Solution {
    public boolean isAnagram(String s, String t) {

      if (s.length() != t.length()) {
        return false;
      }

      Map<Character, Integer> freq = new HashMap<>();
      for (int i = 0; i < s.length(); i++) {
        char sChar = s.charAt(i);
        char tChar = t.charAt(i);

        int sCharFreq = freq.getOrDefault(sChar, 0);
        freq.put(sChar, sCharFreq + 1);

        int tCharFreq = freq.getOrDefault(tChar, 0);
        freq.put(tChar, tCharFreq - 1);
      }

      long results = freq.values().stream().filter(value -> value != 0).count();

      return results == 0;
    }
  }
```

## Time Complexity

O(n)

## Space Complexity

O(1) as the freq contains max 26 keys (english alphabet)
