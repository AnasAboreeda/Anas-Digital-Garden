## Homework (due Tue, Nov 24 EOD)

- Create a new document in your homework folder
    - Name it “Behavioral interview answers - First name Last name.”
    - In this file, write the question “What's an example of a project you're proud of?”
        - Write the answer the way you would answer it in an interview.
        - Remember to use the STARR format
        - Ask someone who knows the project to review your answer and help you improve it
- Copy [this template](https://docs.google.com/spreadsheets/d/1t0zCjJWIqAFR255e15oQ6j9klRJjfg13aibrVqyTrH0/edit?usp=sharing) (modified from the Simple Programmer article)
    - Save this in your homework folder with the name “Behavioral interview story brainstorm - First name Last name”
    - Look at the example sheet to understand how to use the template
    - Fill it out with at least 1 example for each column
